# 🔍 YouTube SEO: Discoverability Engine

This skill optimizes video metadata to ensure the right audience finds the content.

## 🛡️ 1. The SEO Pillars
1.  **Semantic Description**: Use natural language that includes primary and secondary keywords.
2.  **Chapters (Timestamps)**: Improve user experience and Google Search indexing.
3.  **Low-Entropy Tags**: Focus on specific, high-intent tags rather than generic ones.
4.  **CTA Placement**: Strategic placement of links and subscription prompts.

## 🏗️ 2. Industrial SEO Protocol
- **First 2 Lines**: Must contain the most important keywords (visible in search results).
- **Format**: [Summary] -> [Chapters] -> [Resources] -> [Tags].

## ⌨️ Commands
| Command | Action |
| :--- | :--- |
| `/youtube-seo [title]` | Calls `youtube_seo.py` to generate description, chapters, and tags. |
