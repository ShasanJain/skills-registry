# ✍️ YouTube Scripting: Retention Architect

This skill engineers scripts that minimize drop-off and maximize watch time.

## 🛡️ 1. The Script Pillars
1.  **The Hook (0-30s)**: Must validate the thumbnail's promise immediately.
2.  **The Reassurance (30-60s)**: Show the viewer they are in the right place and explain the "stakes."
3.  **The Bridge**: Transition between points using "open loops" (e.g., "But before we get to the secret, we first need to...")
4.  **The Payoff**: Deliver the primary value promised in the title.
