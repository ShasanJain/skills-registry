## 📚 Reference Material

# signatures.md — Reference File

## What This Is
Your sign-off preferences by context. The skill uses this to end every email the right way without asking you each time.

## What To Add
List your preferred closings for different situations. Include:
- The sign-off phrase
- When you use it
- Your name/title as it should appear

## Format
```
## Default (most professional emails)
Sign-off: Warmest Regards,
Name block:
[Your Full Name]
[Title]
[Department, Institution]
[Email]
[Website or ORCID — optional]

## Collegial/peer emails
Sign-off: Best,
Name block: [First name only]

## Quick replies / internal messages
Sign-off: Thanks,
Name block: [First name only]

## Formal / committee correspondence
Sign-off: Best regards,
Name block:
[Full Name], [credentials if relevant]
[Full title and affiliation]

## Student/mentee emails
Sign-off: Best,
Name block: [First name] or [Full name depending on context]
```
