## 📝 Quy trình Thiết lập Hệ thống Blog

Xây dựng nền tảng viết lách mạnh mẽ với hỗ trợ Markdown và SEO đỉnh cao.

### 1. Cấu trúc Nội dung
- Chuyên gia: `documentation-writer`.
- Thiết lập thư mục `/posts` và hệ thống Category/Tag.

### 2. Trải nghiệm Đọc
- Tối ưu Typography, Code Highlighting và Dark Mode.
- Tích hợp Reading Time and Share Buttons.

### 3. Tối ưu GEO/SEO
- Tự động tạo RSS Feed, Sitemap.
- Cấu hình OpenGraph cho mạng xã hội.
